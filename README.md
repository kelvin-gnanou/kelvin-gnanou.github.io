## Deployment

This website is deployed using GitHub Pages.

### Deployment Steps

```bash
1. Push repository to GitHub
2. Open Settings
3. Go to Pages
4. Select:
   Branch: main
   Folder: /root
5. Save
```

GitHub automatically publishes the website.
